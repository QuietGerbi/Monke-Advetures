🐒 Monke Adventures

Monke Adventures — это динамичный 2D-экшен, разработанный на Java с использованием библиотеки Swing. 
Погрузитесь в мир безумных прыжков, сражений с боссами и хардкорного геймплея.
Бедная бибизянка попала зашла в чужой район и ей нужно победить остальных бибизян чтобы выжить и убежать к своей семье.
Благо она нашла посох монаха которым она сможет защититься

Особенности игры

    Классический 2D Геймплей: Плавная анимация и отзывчивое управление в лучших традициях платформеров.

    Эпические боссы: Сразитесь с легендарным Monke Boss — боссом, который не прощает ошибок.

    Кроссплатформенность: игра доступна для Windows, macOS и Linux в виде готовых инсталляторов.
